# Community Event Management System

## 📌 Project Overview

The Community Event Management System is a web-based application developed using HTML, CSS, and JavaScript. It helps users create, manage, and register for events through a simple and user-friendly interface.

This project was developed as a beginner web development project to learn frontend development, local storage, and GitHub deployment.

---

## 🚀 Features

* User Registration
* User Login
* Profile Page
* Profile Photo Upload
* Create Event
* Edit Event
* Delete Event
* Search Events
* Event Categories
* Event Descriptions
* Event Registration
* My Events Page
* Admin Dashboard
* Registration Statistics
* Dark Mode
* Responsive Design
* GitHub Pages Deployment
